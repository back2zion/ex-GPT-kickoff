🎯 핵심 고도화 요소

🧠 지능형 라우팅
• LangGraph 기반 멀티턴 대화
• 컨텍스트 메모리 관리
• 질의 유형별 최적 경로 선택
⚙️ MCP 업무 자동화
• 회의실 예약 + 회의록 자동 작성
• 배차 신청 시스템 연동
• 하이포털 API 통합
🎤 멀티모달 처리
• Florence-2 이미지/도면 분석
• Whisper STT 음성 처리
• 통합 검색 및 인덱싱
🔐 보안 및 권한
• 신규 권한 체계 구축
• 개인화 서비스 제공
• 개인정보 자동 검출
🔍 AS-IS (현재 기능)
대화 및 상호작용
  • 단발성 대화만 지원
    Single-Turn 방식으로 연속적인 대화 불가, 사용자 불만 급증
  • 기본 질의응답
    간단한 검색 기반 답변만 제공, 복잡한 추론 한계
  • 컨텍스트 메모리 없음
    이전 대화 내용을 기억하지 못해 업무 연속성 부족
핵심 기능
  • 번역 서비스
    한영 번역 기능, 사용자 만족도 높음
  • 나만의 AI (기본)
    개인 문서 업로드 및 기본 분석
  • 문서 요약 (제한적)
    기본 템플릿만 제공, 맞춤형 요약 불가
  • 권한 관리 미비
    신규 권한 체계 미구현, 개인화 서비스 불가
시스템 연동 및 자동화
  • MCP 연동 없음
    하이포털, 모바일오피스 등 레거시 시스템 연동 불가
  • 회의실 예약 불가
    외부 시스템 연동 기능 전무
  • 배차 신청 불가
    업무 자동화 기능 미구현
  • 회의록 자동 작성 불가
    STT 기능 미지원
멀티모달 및 고급 기능
  • 이미지/도면 처리 불가
    텍스트 데이터만 처리 가능
  • 음성 인식 미지원
    Whisper STT 기능 없음
  • 문서 비교 기능 없음
    업로드 문서 간 비교 분석 불가
  • 제한적 검색 기능
    기본 벡터 검색만 지원
🚀 TO-BE (목표 기능)
고도화된 대화 및 상호작용
  • 멀티턴 대화
    QWen3-235B, MiniMax-M1(456B), Llama4등 최신 LLM으로 연속적인 대화 지원, 복잡한 업무 요청 처리 가능
  • 지능형 라우팅
    질의 유형에 따른 최적 처리 경로 자동 선택
  • 컨텍스트 메모리 관리
    대화 히스토리 유지, 업무 연속성 보장
확장된 핵심 기능
  • 고급 번역 및 다국어 지원
    한영 자동 번역 + 다양한 언어 지원
  • 개인화된 나만의 AI
    개인별 권한 체계 기반 맞춤형 서비스
  • 지능형 문서 요약
    맞춤형 프롬프트, 용도별 요약 템플릿
  • 고급 문서 비교 분석
    업로드 문서 간 차이점 분석 및 트렌드 파악
MCP 기반 시스템 연동 및 자동화
  • MCP 연동
    검색 기능 위주에서 실행 위주로 변경
  • 회의실 예약
    "내일 오후 2시 중회의실 예약해줘" 자연어 명령 처리
  • 배차 시스템 연동
    출장 목적지 기반 최적 차량 자동 매칭 및 신청
  • Whisper 기반 회의록 자동 작성
    음성 → 텍스트 → 요약 → 메일 전송 자동화
멀티모달 및 AI 고도화
  • Florence-2-Large 기반 비전 처리
    Microsoft의 0.77B 매개변수 통합 비전 모델 - 도면, 설계도 이미지 분석, 캡션 생성, 객체 탐지, OCR 지원
  • 영상/사진 검색
    이미지 콘텐츠 기반 통합 검색 시스템
  • Whisper-Large-v3 기반 음성 처리
    OpenAI의 최신 다국어 음성 인식 모델 - 128 Mel 주파수 빈, 1백만 시간 학습 데이터
  • 고급 문서 검색
    "최근 3년간 하이패스 관련 법령 변경사항" 시계열 분석 및 마크다운 출력 지원
