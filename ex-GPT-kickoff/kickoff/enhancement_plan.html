🎯 핵심 고도화 요소 (월별 단계순)

🧠 1단계: 지능형 라우팅 (7월 15일)

• LangGraph 기반 멀티턴 대화
• 컨텍스트 메모리 관리
• 질의 유형별 최적 경로 선택
• Llama3-70B → QWen3-235B 모델 업그레이드
📊 2단계: 문서 분석 + 경영평가 (8월)

• 지능형 문서 분석 및 비교 (파일 5개)
• 경영평가 보고서 템플릿 제공
• 국정감사 시즌 대비 특화 기능
• 맞춤형 요약 프롬프트 강화
🚷 3단계: 자동화 서비스 통합 (10월)

• 회의록 자동 생성 (STT 기반)
• 회의실 자동 예약 시스템
• 배차 신청 자동화
• 전체 업무자동화 완성
🎤 4단계: 멀티모달 처리 (11월)

• Florence-2-Large 이미지 분석
• Whisper-Large-v3 음성 인식
• 통합 검색 및 인덱싱
• 사진 콘텐츠 검색
🔍 AS-IS (현재 기능)
대화 및 상호작용
  • 단발성 대화만 지원
    Single-Turn 방식으로 연속적인 대화 불가, 사용자 불만 급증
  • 기본 질의응답
    간단한 검색 기반 답변만 제공, 복잡한 추론 한계
  • 컨텍스트 메모리 없음
    이전 대화 내용을 기억하지 못해 업무 연속성 부족
핵심 기능
  • 번역 서비스
    한영 번역 기능, 사용자 만족도 높음
  • 나만의 AI (기본)
    개인 문서 업로드 및 기본 분석
  • 문서 요약 (제한적)
    기본 템플릿만 제공, 맞춤형 요약 불가
  • 권한 관리 미비
    신규 권한 체계 미구현, 개인화 서비스 불가
시스템 연동 및 자동화
  • MCP 연동 없음
    하이포털, 모바일오피스 등 레거시 시스템 연동 불가
  • 회의실 예약/회의록 자동 작성 불가
    외부 시스템 연동 기능 전무 / STT 기능 미지원
  • 배차 신청 불가
    업무 자동화 기능 미구현
멀티모달 및 고급 기능
  • 이미지 처리 불가
    텍스트 데이터만 처리 가능
  • 음성 인식 미지원
    Whisper STT 기능 없음
  • 문서 비교 기능 없음
    업로드 문서 간 비교 분석 불가
  • 제한적 검색 기능
    기본 벡터 검색만 지원
🚀 TO-BE (목표 기능)
고도화된 대화 및 상호작용
  • 멀티턴 대화
    QWen3-235B, Llama4등 최신 LLM으로 연속적인 대화 지원, 복잡한 업무 요청 처리 가능
  • 지능형 라우팅
    질의 유형에 따른 최적 처리 경로 자동 선택
  • 컨텍스트 메모리 관리
    대화 히스토리 유지, 업무 연속성 보장
확장된 핵심 기능
  • 고급 번역 및 다국어 지원
    한영 자동 번역 + 다양한 언어 지원
  • 개인화된 나만의 AI
    개인별 권한 체계 기반 맞춤형 서비스
  • 지능형 문서 요약 + 경영평가 템플릿
    맞춤형 프롬프트, 용도별 요약 템플릿 + 경영평가 보고서 자동 생성 지원 (8월 구현)
  • 고급 문서 비교 분석
    파일 5개까지 동시 업로드, 문서 간 차이점 분석 및 트렌드 파악
MCP 기반 시스템 연동 및 자동화
  • MCP 연동
    검색 기능 위주에서 실행 위주로 변경
  • 회의실 예약 + 회의록 자동 작성
    "회의실 예약해줘" → 예약 → 회의 진행 → 회의록 자동 작성 및 메일 전송까지 통합 프로세스
  • 배차 신청 자동화
    출장 목적지 기반 최적 차량 자동 매칭 및 신청 (10월 구현)
멀티모달 및 AI 고도화
  • Florence-2-Large 기반 비전 처리
    비전 모델 - 이미지 분석, OCR 지원
  • Whisper-Large-v3 기반 음성 처리
    다국어 음성 인식 모델, 1백만 시간 학습 데이터
  • 고급 문서 비교 기능
    다중 파일 동시 업로드, 문서 간 차이점 분석
  • 지능형 문서 검색
    시계열 분석 및 마크다운 출력 지원

핵심 기능 비교 요약

기능 영역 AS-IS (현재) TO-BE (목표) 구현 시기

시스템 연계 연동 없음 하이포털 등 관련시스템 연계 9월

• 7월 15일: QWen3-235B 기반 멀티턴 대화 MVP 오픈
• 8월: 지능형 문서 분석 + 경영평가 보고서 템플릿 (국감대비)
• 9월: 하이포털 등 관련 시스템 연계 작업 (MCP API 구축)
• 10월: 회의록 자동생성 + 회의실 예약 + 배차 자동화 통합
• 11월: Florence-2 + Whisper-v3 멀티모달 완성
